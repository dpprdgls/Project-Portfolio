About Me

Hello there! My name is Colin Douglas. I am a photographer, graphic-design dabbler, architectural enthusiast, traveler, reader, racing-driver, mechanic, hiker, gamer, cyclist, and budding full-stack engineering specialist. I currently work in the automotive manufacturing field within the maintenance department, specializing in laser welding. Within this field I have gained a great deal of knowledge about integrated systems, automation, as well as optimization and the application of the LEAN manufacturing ideology. I have studied the Six Sigma principles that help to achieve efficiency within processes while maintaining minimal defects. I enjoy the statistical analysis of data and learned to apply this knowledge through the use of the python coding language. I am always interested in learning and growing which is why I enrolled in the UC Berkeley EdX coding bootcamp.
This bootcamp challenged me to create multiple projects to demonstrate my knowledge in HTML, CSS, JavasScript, JS.Node, and React languages. I have created this website with my new skills to showcase my completed projects. It also features links to my work on GitHub and my LinkedIn profile so that future employers or collaborators can view my code more in-depth and contact me. This website will be periodically updated as my skills and project counts progress.
